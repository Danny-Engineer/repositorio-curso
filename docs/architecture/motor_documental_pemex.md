# Motor Documental PEMEX

## Flujo objetivo

1. Subir oficio PDF.
2. Extraer texto.
3. Analizar asunto, destinatario, urgencia y riesgos.
4. Consultar RAG documental.
5. Generar borrador institucional.
6. Completar plantilla DOCX.
7. Exportar documento listo para revisión.

## v4

Esta versión incluye análisis y generación de respuesta mediante texto pegado manualmente.
